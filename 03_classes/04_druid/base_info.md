# ドルイド

## ドルイドの基本特性

| 項目 | 内容 |
|------|------|
| **主要能力** | 判断力 |
| **ヒット・ポイント・ダイス** | ドルイド・レベルごとにD8 |
| **セーヴィング・スロー習熟** | 知力と判断力 |
| **スキル習熟** | 以下から2つ選択：動物使い、魔法学、看破、医術、自然、知覚、宗教、生存 |
| **武器習熟** | 単純武器 |
| **道具習熟** | 薬草師道具 |
| **防具習熟** | 軽装鎧、盾 |
| **開始装備** | AまたはBを選択：(A) レザーアーマー、盾、鎌、ドルイド焦点具（クォータースタッフ）、探検家パック、薬草師道具、9GP、または (B) 50GP |
