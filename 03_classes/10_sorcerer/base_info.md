# ソーサラー

## ソーサラーの基本特性

| 項目 | 内容 |
|------|------|
| **主要能力** | 魅力 |
| **ヒット・ポイント・ダイス** | ソーサラー・レベルごとにD6 |
| **セーヴィング・スロー習熟** | 耐久力と魅力 |
| **スキル習熟** | 以下から2つ選択：魔法学、ぺてん、看破、威圧、説得、宗教 |
| **武器習熟** | 単純武器 |
| **防具習熟** | なし |
| **開始装備** | AまたはBを選択：(A) スピア、ダガー×2、アーケイン焦点具（クリスタル）、ダンジョン探検家パック、28GP、または (B) 50GP |
