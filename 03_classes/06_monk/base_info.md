# モンク

## モンクの基本特性

| 項目 | 内容 |
|------|------|
| **主要能力** | 敏捷力と判断力 |
| **ヒット・ポイント・ダイス** | モンク・レベルごとにD8 |
| **セーヴィング・スロー習熟** | 筋力と敏捷力 |
| **スキル習熟** | 以下から2つ選択：軽業、運動、歴史、看破、宗教、隠密 |
| **武器習熟** | 単純武器と軽特性を持つ軍用武器 |
| **道具習熟** | 職人道具または楽器から1種類を選択（「装備」参照） |
| **防具習熟** | なし |
| **開始装備** | AまたはBを選択：(A) スピア、ダガー×5、上記で選択した職人道具または楽器、探検家パック、11GP、または (B) 50GP |
