# バーバリアン

## バーバリアンの基本特性

| 項目 | 内容 |
|------|------|
| **主要能力** | 筋力 |
| **ヒット・ポイント・ダイス** | バーバリアン・レベルごとにD12 |
| **セーヴィング・スロー習熟** | 筋力と耐久力 |
| **スキル習熟** | 以下から2つ選択：動物使い、運動、威圧、自然、知覚、生存 |
| **武器習熟** | 単純武器と軍用武器 |
| **防具習熟** | 軽装鎧、中装鎧、盾 |
| **開始装備** | AまたはBを選択：(A) グレートアックス、ハンドアックス×4、探検家パック、15GP、または (B) 75GP |
