# ローグ

## ローグの基本特性

| 項目 | 内容 |
|------|------|
| **主要能力** | 敏捷力 |
| **ヒット・ポイント・ダイス** | ローグ・レベルごとにD8 |
| **セーヴィング・スロー習熟** | 敏捷力と知力 |
| **スキル習熟** | 以下から4つ選択：軽業、運動、ぺてん、看破、威圧、捜査、知覚、説得、手先の早業、隠密 |
| **武器習熟** | 単純武器と妙技または軽特性を持つ軍用武器 |
| **道具習熟** | 盗賊道具 |
| **防具習熟** | 軽装鎧 |
| **開始装備** | AまたはBを選択：(A) レザーアーマー、ダガー×2、ショートソード、ショートボウ、アロー×20、矢筒、盗賊道具、盗賊パック、8GP、または (B) 100GP |
